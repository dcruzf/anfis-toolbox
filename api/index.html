    üéØ Membership Functions¶

    Complete set of 13 fuzzy membership function implementations:

    Function Type Parameters Module
    GaussianMF Gaussian mean, sigma membership-functions
    Gaussian2MF Two-sided Gaussian sigma1, c1, sigma2, c2 membership-functions
    TriangularMF Triangular a, b, c membership-functions
    TrapezoidalMF Trapezoidal a, b, c, d membership-functions
    BellMF Bell-shaped a, b, c membership-functions
    SigmoidalMF Sigmoidal a, c membership-functions
    DiffSigmoidalMF Difference of sigmoids a1, c1, a2, c2 membership-functions
    ProdSigmoidalMF Product of sigmoids a1, c1, a2, c2 membership-functions
    SShapedMF S-shaped a, b membership-functions
    LinSShapedMF Linear S-shaped a, b membership-functions
    ZShapedMF Z-shaped a, b membership-functions
    LinZShapedMF Linear Z-shaped a, b membership-functions
    PiMF Pi-shaped a, b, c, d membership-functions
